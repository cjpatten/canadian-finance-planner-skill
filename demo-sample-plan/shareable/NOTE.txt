This folder would contain PDF and Excel versions of your financial plan.

In a live session, Claude generates these automatically:
  - My-Financial-Profile.pdf
  - My-Budget.pdf
  - My-Plan.pdf
  - My-Budget.xlsx (editable spreadsheet with formulas)

These are not included in the demo because they require Python libraries
(weasyprint, openpyxl) that run during a live Cowork session.

To see them generated, install the skill and run it with Claude!
